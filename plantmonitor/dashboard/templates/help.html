Help & Documentation

Getting Started

Dashboard Overview

The dashboard displays real-time sensor data from your plant monitoring system. All sensor readings are updated automatically every 30 seconds.

Navigation

The system has five main sections:

  • Dashboard: View real-time sensor data
  • Analytics: View historical data trends
  • Sensors: Detailed information about each sensor
  • Options: Configure system settings and thresholds
  • Help: This documentation

Sensor Data

Available Sensors

The system monitors the following parameters:

  • Temperature (Â°C)
  • Pressure (hPa)
  • Soil Moisture (%)
  • Humidity (%)
  • Light Level
  • CO2 Level (ppm)
  • Battery Level (V)
  • WiFi Signal Strength (dBm)
  • GPS Location

Data Updates

The system automatically updates sensor data every 30 seconds. You can see the last update time displayed on the dashboard.

Configuration

Setting Thresholds

You can configure various thresholds in the Options section:

  • Soil Moisture Threshold (0-100%)
  • Temperature Threshold (Â°C)
  • Humidity Threshold (0-100%)
  • Light Level Threshold (0-10)
  • Update Interval (30-3600 seconds)
  • Alert Threshold (1-10)

Saving Changes

After modifying any settings, click the "Save Settings" button to apply your changes. A success message will appear when the settings are saved successfully.

Troubleshooting

Common Issues

  • If data is not updating, check your internet connection
  • If sensors show incorrect values, try recalibrating them
  • If the system is not responding, check the battery level
  • If WiFi signal is weak, consider moving the device closer to the router

Error Codes

The system uses error codes to indicate specific issues:

  • 0: No error
  • 1: Sensor reading error
  • 2: Communication error
  • 3: Battery low
  • 4: WiFi connection lost
